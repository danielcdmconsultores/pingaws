# pingaws
Ping all AWS regions from your ip.

[go to this web: ](https://danielcdmconsultores.github.io/pingaws/)

## Librerías y Servicios Utilizados

- **[jQuery 4.0.0](https://jquery.com/)**: Utilizado para la manipulación del DOM, gestión de eventos y lógica de la aplicación.
- **[L2.io IP API](https://l2.io/)**: Servicio utilizado para obtener y mostrar la dirección IP pública del usuario.
- **[LinkedIn Profile Badges](https://platform.linkedin.com/badges/)**: Integración de perfiles profesionales en el pie de página.
- **[AWS DynamoDB Endpoints](https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/Endpoints.html)**: Utilizados como puntos finales de prueba para medir la latencia hacia las diferentes regiones de Amazon Web Services.
